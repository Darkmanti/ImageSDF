# Image SDF
Unity plugin for importing SVG icons and rendering them usinf Signed Distance Field's

## Getting Started
Download latest `.unitypackage` from releases and import to your Unity project.

## Support
- Runtime - all platforms
- Editor
  |   OS                |  Supported   |
  |---------------------|--------------|
  | Windows x64         |    ✓        |
  | MacOS (x64 + arm64) |    ✓        |
  | Linux               | In Progress  |
